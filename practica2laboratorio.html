- Materiales -

  • 1 Computadora
  • 3 Dispositivos distintos
  • Cables de red directos y cruzados

- Objetivos Específicos -

  • Conecta tu computadora a la red de la universidad mediante la conexión wifi
  • Identifica la Ip de acceso a la red de tu computadora.
  • En tu computadora instala Xampp para configurar un servidor local, habilita los servicios de Apache y monta un sitio web: https://www.apachefriends.org/es/index.html
  • Habilita el acceso al puerto 80 desde el firewall del equipo configurado como el servidor local.
  • Comprueba el acceso al sitio web montado en el servidor local desde 3 dispositivos distintos (para acceder es necesario estar en la misma red).
  • Elabora un mapa de accesos de equipos y sus respectivas IPs en la Red de las computadoras de tus demás compañeros.
  • Repite los mismos pasos ahora utilizando el cable directo y conectándote a la red de la universidad cableada.
  • Repite los mismos pasos, pero ahora con el cable cruzado entre dos equipos, configurando la IP de ambos equipos a 192.168.0.1 (un equipo) y el 192.168.0.2 (el segundo equipo).

    • Construcción

    En esta práctica se experimenta con la publicación local de un sitio web y métodos para acceder al mismo siempre que los dispositivos estén en la misma red, asi como el uso del programa XAMPP.

    En esta primera imagen se muestra un mapa de accesos, en el cual se puede ver el acceso al sitio web en el puerto 80 desde la misma red inalámbrica.

    En siguiente imagen se muestra un mapa de accesos, en el cual se puede ver el acceso al sitio web por medio de conexión directa a la red universitaria.

    En la última imagen se muestra un mapa de accesos, en el cual se puede ver el acceso al sitio web con cable cruzado configurando las IPs.

    - Resultados -

    Acceso al sitio web en el puerto 80 desde la misma red inalámbrica.

    Acceso al sitio web por medio de conexión directa a la red universitaria.

    Acceso al sitio web con cable cruzado configurando ips.

    - Conclusiones -

    Al finalizar la práctica aprendimos a identificar la IP de nuestras computadoras, para realizar y configurar una red local, y conectarnos a esta a través de distintos dispositivos.
